Yet another C++ logging library.

Fast

Blackhole is designed to be the fastest logger in C++ ever. This includes dark template magic inside, but for the price of it you are not paying for things you don't use.

Blackhole benchmarks →

Configurable

You can configure everything: severity levels, filtering, exception handling, formatting and configuration itself. Settings can be specified both in source code and from file.

Configuration example →

Extensible

You are not restricted with predefined formatters, sinks, frontends or initializers. You can even change formatter-sink connectivity interface with the template specialization.

How to extend →

Header only

No runtime dependencies! Just include and develop. Once again, you are not paying for things you don't use. Don't need syslog sink? Don't include and compile it!

Advantages →

Attribute-based

Every log event contains multiple attributes. In Blackhole log attributes is everything. It can be indexed, it can be filtered and it is type-safe.

More on attributes →

Robust

This project is developed using TDD and CI for every operating system it is meant to be build, so any found bugs is fixed without breaking existed functional.

Build status →

Install manually.

Manual Install Instructions

~ $ git clone https://github.com/3Hren/blackhole.git

~ $ cd blackhole

~/blackhole $ cmake

~/blackhole $ make install

# Now Blackhole is installed
